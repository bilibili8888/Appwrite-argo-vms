#### 提示:

1.不要选择nodejs模板，选择导入仓库部署,仓库index.js里面修改参数

2.部署命令改成npm install

3.进入appwrite网站设置里面，找到Joye打开Any权限，设置时间最大900


4.redeploy,然后点excute

5.打开网址查看是否出现网页，出现则部署正常

6.把网页定时访问保活，推荐利用下面项目部署保活，也可以利用哪吒监控保活

#### 进阶玩法：可以部署自己的socks5节点，使用脚本 https://github.com/dsadsadsss/java-wanju.git

然后复制节点到v2rayn中，把用户名，密码找到

按照格式socks5://用户名:密码@ip:端口格式填入nodejs里面
